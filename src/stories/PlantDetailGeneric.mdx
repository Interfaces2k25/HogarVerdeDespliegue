# PlantDetailGeneric

El componente **PlantDetailGeneric** se utiliza para mostrar el detalle completo
de una planta seleccionada dentro del catálogo.

Presenta información relevante como la imagen, nombre, precio, descripción e
información adicional, permitiendo al usuario consultar el producto de forma clara
y estructurada.
